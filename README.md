# QR Code Generator

A single-file HTML tool for generating customizable QR codes. No server required - just open in a browser.

## Features

- **Generate QR Codes** - Enter any text or URL to create a QR code
- **Adjustable Size** - Slider control from 100px to 400px
- **Custom Colors** - Pick foreground and background colors
- **Error Correction Levels** - Choose L, M, Q, or H recovery levels
- **Download as PNG** - Save QR codes to your device
- **Copy to Clipboard** - One-click copy for easy sharing
- **History** - Recent QR codes saved locally for quick access

## Usage

1. Open `index.html` in any modern browser
2. Enter text or URL
3. Adjust size, colors, and error correction as needed
4. Click "Generate QR Code"
5. Download or copy the result

## Technologies

- Vanilla HTML, CSS, JavaScript
- [QRCode.js](https://github.com/davidshimjs/qrcodejs) via CDN
- localStorage for history persistence

## Browser Support

Works in all modern browsers (Chrome, Firefox, Safari, Edge).
